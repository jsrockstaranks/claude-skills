# Example Input

Build a `Badge` component for a design system. Requirements:

- Variants: `default`, `success`, `warning`, `error`, `info`
- Sizes: `sm`, `md`, `lg`
- Optional leading icon (accepts any ReactNode)
- Optional dismiss button (X) that calls an `onDismiss` callback
- Should work as both a controlled and uncontrolled component
- Use Tailwind CSS
